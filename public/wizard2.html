Instructions 

This spreadsheet provides a framework to evaluate components
(e.g. applications, services, modules) for hosting in either a public or private cloud.

Evaluation Criteria Worksheet

This worksheet lists all of the evaluation criteria used within the framework.  Each criteria has a unique name, a description, and a list of options that can be selected for the criteria when scoring components for that criteria.  Each of the options provides a suitability score for both public and private cloud hosting.  A higher score indicates greater suitability for cloud hosting.

Typical Customizations

Changing criteria scoring  

This is the simplest customization allowing the relative scoring of the options provided to be modified. 

Changing the options

This modification changes the options listed for the criteria.  Adding additional options or removing existing options will require modifications to the Components worksheet.  Renaming options but keeping the same number of options will not

Adding a new evaluation criteria

This modification includes new criteria in the suitability scoring.  This modification will require changes to Weighting and Components worksheets

Note: removing an existing criteria is not recommended

It is easier to put a zero weight in the Weighting worksheet to effectively remove a criteria from consideration

Typical Customizations

Modifying criteria weights  

This allows some criteria to have more importance in the final analysis than other criteria.  Two predefined weighting profiles (Coordination Operating Model, Compliance Focus) are given as examples.  To use either of these profiles, copy the weighting factors in the corresponding column and paste them into the Weighting column, or set custom weighting for each criteria according to customer priorities.  You can also reflect the customer bias toward public or private cloud by setting the Private Cloud Bias - greater than 1 (e.g., 1.1) reflects a bias toward Private Cloud, less than 1 (e.g., 0.9) reflects a bias toward Public Cloud

Weighting Worksheet

This worksheet provides the ability to weight each of the evaluation criteria in the Criteria worksheet.  Setting a criteria weight to zero effectively removes that criteria from consideration.
